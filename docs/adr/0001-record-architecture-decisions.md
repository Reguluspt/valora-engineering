# ADR 0001 — Record Architecture Decisions

## Status

Accepted

## Context

Valora Design Book v1.2-final is the implementation source of truth. During implementation, ambiguities may appear.

## Decision

Use ADRs for implementation decisions that clarify but do not contradict the Design Book.

## Consequences

- Every meaningful deviation requires ADR.
- Domain changes require Design Change Request.
- Codex must not invent domain behavior.
